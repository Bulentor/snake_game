# snake_game

Описание проекта
Классическая игра «Змейка» с расширенным геймплеем, реализованная на чистом JavaScript. В игру встроены механики автоматического ускорения, система уровней и препятствий, а также возможность проходить сквозь границы экрана. Прогресс игрока (лучший результат) сохраняется локально в браузере.

Стек технологий
Frontend: HTML5 (Canvas API), CSS3, JavaScript (Vanilla JS).
Backend: отсутствует (клиентское приложение).
Database: LocalStorage (для хранения рекордов).

Как запустить локально
Скопируйте код игры в файл с названием index.html.
Откройте этот файл в любом современном браузере (Chrome, Firefox, Safari) двойным кликом мыши.
Для управления используйте стрелки на клавиатуре.
